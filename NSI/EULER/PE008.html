Trouver les treize chiffres adjacents dans ce nombre à 1000 chiffres qui donnent le plus grand produit. Quelle est la valeur de ce produit ?

On donne deux solutions pour la lecture des données, et aussi des variantes dans un style fonctionnel.

nombre_chaine = gros_nombre.replace("\n", "")
tableau_chiffres = [int(chiffre) for chiffre in nombre_chaine]
# variante fonctionnelle de la ligne 23 :
# tableau_chiffres = list(map(int, nombre_chaine))

# variante des lignes 22-23 sans le "replace"
tableau_chiffres = []
for caractère in gros_nombre:
	if caractère != "\n":
		tableau_chiffres.append(int(caractère))


longueur = len(tableau_chiffres)


def produit13(i):
	"""Retourne le produit de 13 chiffres consécutifs,
	à partir du rang i, dans `tableau_chiffres`
	"""
	assert i + 13-1 < longueur, "danger sortie du tableau"
	ans = 1
	for x in range(13):
		ans = ans * tableau_chiffres[ i+x ] # i+ 13-1 < longueur !!!
	return ans

maxi = 0
for i in range(longueur - 13):
	p = produit13(i)
	if p > maxi:
		maxi = p

print(maxi)
